## 자주 쓰는 기본 명령어
- 💡 모든 명령어 뒤에 `--help` 옵션을 주면 자세한 사용 방법이 나온다.
- ex) `ls --help`  

### pwd(print working directory)
- 현재 작업 중인 디렉토리 정보 출력. 

```
$ pwd
/home/itholic
```

### cd(change directory)
- 경로 이동
