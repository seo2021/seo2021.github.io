## 변수의 scope
- 변수가 선언된 블럭이 그 변수의 사용범위이다.
```java
public class VariableScopeExam {  
  int globalScope = 10; // 인스턴스 변수  
  public void
}
```

## 정리
- 단항, 이항, 삼항 연산자 순으로 우선순위를 갖는다.
- 산술, 비교, 논리, 대입 연산자 순으로 우선순위를 갖는다.
- 단항과 대입 연산자를 제외한 모든 연산 방향은 왼쪽에서 오른쪽이다.
