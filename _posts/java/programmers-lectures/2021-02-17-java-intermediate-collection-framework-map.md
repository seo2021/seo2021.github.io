## Map 컬렉션 클래스
- Map 인터페이스는 Collection 인터페이스와는 다른 저장 방식을 가진다.
- Map 인터페이스를 구현한 Map 컬렉션 클래스들은 **키와 값을 하나의 쌍으로 저장하는 방식(key-value 방식)**을 사용.
  - 여기서 키(key)란 실질적인 값(value)을 찾기 위한 이름의 역할을 한다.
